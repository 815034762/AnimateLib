# AnimateLib
一个练习动画的自定义View，里面有一些可以根据TextView宽度自动调节文字大小的View，类似广告闪烁的View
![image](https://github.com/815034762/AnimateLib/blob/master/capture/ezgif.com-video-to-gif.gif)

### 用法 | Usage
 1. 宽度固定，字体自适应控件大小的TextView
 2. 类似广告灯闪烁的View
 3.从左到右滚动的View
